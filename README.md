Este repositório contém a infraestrutura moderna do e-commerce Marluce Cunha Pijamas, desenvolvida com tecnologias de ponta para oferecer uma experiência de compra rápida, responsiva e elegante. O projeto utiliza agentes de IA para garantir que a interface seja fiel ao design original, convertendo elementos visuais em um código Next.js limpo e performático.

🚀 Tecnologias Utilizadas
Next.js 15+ — App Router, React 19 e TypeScript (Strict Mode).

Tailwind CSS v4 — Estilização moderna com design tokens em OKLCH.

shadcn/ui — Componentes de interface acessíveis e altamente customizáveis.

Lucide React — Biblioteca de ícones vetoriais.

🛠️ Como Iniciar o Projeto
Clonar o repositório

Bash
git clone https://github.com/AlexanderBits/Marluce-Cunha-Pijamas.git
cd Marluce-Cunha-Pijamas
Instalar dependências

Bash
npm install
Ambiente de Desenvolvimento

Bash
npm run dev
🏗️ Estrutura do Projeto
src/
  app/             # Rotas e layouts do Next.js
  components/      # Componentes React (UI e Seções)
    ui/            # Primitivos shadcn/ui
  lib/             # Utilitários e funções auxiliares
  types/           # Definições de TypeScript
public/
  images/          # Fotos dos produtos e banners
  seo/             # Favicons e metadados de compartilhamento
docs/              # Documentação de design e pesquisa
📋 Comandos Disponíveis
npm run dev: Inicia o servidor de desenvolvimento em localhost:3000.

npm run build: Gera a versão otimizada para produção.

npm run lint: Executa a verificação de erros no código.

npm run typecheck: Valida a integridade dos tipos TypeScript.

🐳 Docker (Opcional)
Para rodar o ambiente utilizando containers:

Bash
docker compose up dev --build
